# avitoTask

Cервис, хранящий пользователя и сегменты, в которых он состоит (создание, изменение, удаление сегментов, а также добавление и удаление пользователей в сегмент)


## Как поднять сервис:

```
docker-compose up --build my-app
```

## Как удалить сервис:

```
docker-compose down
```
